# Sistema de Cadastro e Login 

Este projeto é um sistema simples de cadastro. Ele permite criar uma conta, fazer login, alterar sua senha e deletar sua conta.

Ele foi criado com fins educacionais e serve para praticar conceitos básicos de programação, autenticação de usuários e uso de banco de dados de forma prática.

## Funcionalidades: 

- Criar uma conta

- Fazer o Login

- Alterar sua senha

- Deletar sua conta

Caso não exista o cadastro, o sistema informa ao usuário.

## Pré-requisitos:

- Python 3.13 instalado
- Biblioteca SQLite3 (normalmente já vem junto com o python)
- Biblioteca bcrypt (Para instalar digite, `py -m pip install bcrypt` no seu terminal.)
- Biblioteca time (já vem junto com o python)

## Como executar o projeto:

1 - Clone o repositório:
   
   git clone https://github.com/caio180/Projetos-em-python.git

2 - Acesse a pasta do projeto:
   
   cd Projetos-em-python/Projetos_python_Sql/Sistema_de_cadastro

3 - Execute o arquivo principal:
   
   python main.py
