# Agenda de contatos 
Este projeto é um sistema bem simple de agenda de contatos. Ele permite adicionar contatos, listar os contatos cadastrados e buscar um contato.

Este projeto foi criado para fins educacionais, com o objetivo de praticar conceitos básicos de programação.

 
## Funcionalidades:
- Adicionar contatos

- Listar os contatos cadastrados

- Buscar um contato

E caso não existam os contatos, o sistema informa ao usuário.

## Pré-requisitos:

- Python 3.13 instalado

## Como executar o projeto:

1 - Clone o repositório:
   
   git clone https://github.com/caio180/Projetos-em-python.git

2 - Acesse a pasta do projeto:
   
   cd Projetos-em-python/Projetos_com_arquivos/Lista_contatos

3 - Execute o arquivo principal:
   
   python main.py
